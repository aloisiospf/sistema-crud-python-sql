# Sistema CRUD com Python e SQLite

Projeto simples de cadastro de usuários com operações de criação, leitura, atualização e exclusão de dados.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Atualização de registros
- Exclusão de usuários

## Tecnologias
- Python
- SQLite

## Como executar

```bash
python main.py
```

## Descrição
Este projeto foi desenvolvido para praticar lógica de programação, manipulação de banco de dados e operações CRUD utilizando Python e SQLite.
